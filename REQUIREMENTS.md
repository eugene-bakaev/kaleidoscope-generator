# Kaleidoscope generator

We're going to create a kaleidoscope, similar to the one used in iOS for wallpaper

First of all we will need to create some random svg image from a set of "primitives"
List of primitives

- circles
- concentric circles (folded circles)
- spirals
- zigzags
- lines
- dots
- polygons (3-5)
- sines
  On a some background

both backgrounds and primitives should be colored

On the builder page user should be able to

- generate base image by customizing following options
  - select list of primitives
  - select palette
- set options for result kaleidoscope image
  - number of sectors
  - particular part of a base image which will be replicated in sectors
    - user should be able to rotate, flip, change palette, scale, and change a particuar part of the image which will be used for kaleidoscope
